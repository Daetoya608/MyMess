Запрос websocket (header) от клиента к серверу:

{
    "token": str,
}

-------------------------------------------------------------

содержание jwt-токена:

{
    "id": int,
    "username": str,

}

--------------------------------------------------------------

Ответ сервера клиенту по websocket

{
    "sender_id": int,
    "receiver_id": int,
    "content": dict = {
        "text_content": str,
    },

}

---------------------------------------------------------------

Ответ клиента серверу на успешное получение данных
"received"

---------------------------------------------------------------

Формат отправки данных клиента серверу (устарело)

{
    "token": str,
    "message": {
        "sender_id": int,
        "receiver_id": int,
        "content": {
            "text_content": str,
        }
    }
}

---------------------------------------------------------------

Формат отправки данных клиента серверу

{
    "sender_id": int,
    "receiver_id": int,
    "content": {
        "text_content": str
    }
}
