Запрос websocket (header) от клиента к серверу:

{
    "token": str,
}

-------------------------------------------------------------

содержание jwt-токена:

{
    "id": int,
    "username": str,

}

--------------------------------------------------------------

Ответ сервера клиенту по websocket

{
    "sender_id": int,
    "receiver_id": int,
    "content": dict = {
        "text_content": str,
    },

}

---------------------------------------------------------------
